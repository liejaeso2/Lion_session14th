# 시도별 실시간 대기질 정보 조회

## 프로젝트 소개

공공데이터포털의 한국환경공단 에어코리아 대기오염정보 API를 활용하여 시도별 실시간 대기질 정보를 조회하는 React 프로젝트입니다.

## 사용 기술

- React
- Vite
- Axios
- useState / useEffect
- Tailwind CSS
- 공공데이터포털 Open API

## 주요 기능

- 시도 선택 기능
- axios GET 요청으로 공공데이터 API 연동
- 미세먼지, 초미세먼지, 오존, 통합대기환경지수 출력
- 로딩 상태 표시
- 에러 상태 표시
- Tailwind CSS를 이용한 카드형 UI 구성

## 실행 방법

```bash
npm install
npm run dev
